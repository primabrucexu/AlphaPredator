# AlphaPredator 当前实际进度（供 AI agent 读取）

## 文档用途

- 本文件只记录 **已经实际落地到仓库** 的阶段完成度，供 AI agent 快速读取当前项目状态。
- `phase.md` 维护阶段目标；`progress.md` 维护实际完成度、当前判断、下一步。
- 进度判断以当前代码、当前运行结果、当前已验证事实为准，不把计划写成已完成。

## 维护规则

- 每次完成一轮实际功能推进后，必须同步更新 `progress.md`。
- 每次更新只需要维护以下固定字段：
  - `当前推进功能`
  - `状态`：`未开始` / `进行中` / `已完成`
  - `是否完成`：`是` / `否`
  - `当前判断`
- 不需要记录本次改动涉及哪些文件。
- 不保留冗长变更历史；只保留当前真实状态，必要时更新“最近一次更新”。
