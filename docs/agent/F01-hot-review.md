# F01：热点复盘

## 背景

- 主要来源：韭研公社的复盘图片，包含热点板块、题材、连板等信息。
- 通过韭研公社 API 自动获取当日涨停复盘信息和图片，接口文档见 [jygs-api.yml](../human/api-docs/jygs-api.yml)。
- 韭研公社需要登录后访问，系统需要在数据初始化页面提供登录入口，并保存可复用的登录凭据。

## 目标

- 建立热点复盘图片抓取、解析、存储、查询与展示链路。
- 任务模式与市场数据拉取保持一致：支持任务创建、启动、进度跟踪、失败重试与终止。
- 支持指定日期范围的全量获取，以及按指定交易日手动触发增量更新。
- 在页面展示热点板块、题材、连板、复盘图片，并支持个股行情联动。
- 支持多交易日热点变化对比。
- 为后续 AI 学习用户选股偏好提供热点板块、题材、连板等输入数据。

## 不做什么

- 不在本文档中定义数据库表结构，数据模型以 [AlphaPredator.dbml](../human/data-model/AlphaPredator.dbml) 为准。
- 不在本文档中维护韭研公社完整 API 细节，接口细节以 [jygs-api.yml](../human/api-docs/jygs-api.yml) 为准。

## 设计方案

- 数据抓取：
  - 全量获取时，与市场数据初始化一致，支持指定日期范围。
  - 增量更新时，支持按指定交易日手动触发。
  - 每次获取前检查韭研公社登录凭据是否可用；不可用时提示重新登录。
- 数据解析：
  - 从复盘图片中提取热点板块、题材、连板信息与个股信息。
  - API 文档未指出且当前项目不需要的字段不进入主链路。
- 页面展示：
  - 首页提供热点复盘模块入口。
  - 用户可进入详情查看当日热点复盘信息和图片。
  - 复盘模块中的个股名称可跳转到个股详情页。
  - 展示每个交易日的热点板块列表，并支持多交易日热点变化对比。
  - 对比方式为折线图展示某个板块在多个交易日内每天的涨停家数变化，并允许通过点击图标过滤。
  - 默认展示近 5 个交易日，提供近 3 个交易日、近 10 个交易日、近 20 个交易日快捷选择。

## 数据与接口依赖

- 韭研公社 API 文档：[jygs-api.yml](../human/api-docs/jygs-api.yml)。
- 数据模型：[AlphaPredator.dbml](../human/data-model/AlphaPredator.dbml)。
- 韭研公社直接访问地址：https://www.jiuyangongshe.com/。

## 验收标准

- 用户可以创建并启动热点复盘抓取任务。
- 系统可以按日期范围抓取复盘图片和对应信息。
- 系统可以解析并存储热点板块、题材、连板与个股信息。
- 页面可以展示当日热点复盘信息、复盘图片和多交易日热点变化。
- 页面中的个股可以跳转到个股详情页。
- 登录凭据不可用时，系统能提示用户重新登录。
