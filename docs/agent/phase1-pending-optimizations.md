# Phase 1 待优化项

> 本文件记录 Phase 1 已识别但暂不阻塞达成标准的优化点，供后续迭代参考。

## 待优化项 1：分时图

- **所属阶段**：Phase 1（功能增强）
- **当前状态**：`StockDetailPage` 中的"分时"Tab 已占位，标注 `disabled: true`，显示"分时图功能开发中"。
- **背景**：日线 K 线图已完整实现，分时图是常规行情软件的标配视图，但 Phase 1 达成标准未明确要求，故暂缓。
- **待做工作**：
    - 后端提供分时行情数据接口（含分钟级 OHLCV 或逐笔聚合）。
    - 前端 `StockDetailPage` 启用并实现分时图 Tab（价格折线 + 成交量图 + 均价线）。
- **风险**：Tushare 分钟线接口调用频次受限，需评估数据源可行性（pro.stk_mins 或 minutes_indicator）。
- **验收**：分时 Tab 可展示当日完整分时价格与量能曲线。

## 待优化项 2：FocusPage（重点关注）真实功能

- **所属阶段**：Phase 1（功能增强）
- **当前状态**：`FocusPage.tsx` 有骨架页面，数据完全硬编码，标注"Phase 1 占位"。
- **背景**：Phase 1 关键任务未明确要求自选股功能，当前占位页面仅用于展示导航完整性。
- **待做工作**：
    - 设计 SQLite 自选股表（`watchlist`：用户手动添加的股票列表 + 备注）。
    - 后端提供自选股增删查接口。
    - 前端 `FocusPage` 接入真实接口，支持添加/移除股票，实时展示价格与涨跌。
- **风险**：依赖 Phase 1 的股票搜索能力，功能边界与 Phase 3"选股模式"存在一定重叠，需在设计时区分"人工关注"与"AI 推荐"。
- **验收**：用户可在 FocusPage 添加/移除自选股，并查看实时行情摘要。
